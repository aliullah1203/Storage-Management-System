# Storage Management System
A secure and user-friendly backend system to manage, organize and share files, notes, and documents. Built with Node.js, Express.js, and MongoDB following a modular design pattern.


# Features
User Authentication:
- Signup / Register
- Login
- Forgot password & Reset password
- Google OAuth login

File Management:
- Upload files (any type)
- Copy files
- Rename files
- Duplicate files
- Delete files
- Mark / Unmark as Favorite
- Lock / Unlock files for privacy
- Share files with other users

Note Management:
- Create and manage notes
- Attach notes to folders

Folder Management:
- Organize files, images, and notes
- View files by folder

Favorites:
- Quickly access favorite files

Sharing:
- Share files with other users or external links

Calendar & Filtering:
- Filter files by date
- View files in a calendar view

File Usage:
- Track storage usage
- Monitor used vs available storage

User Profile:
- Update profile (name, email, profile picture)
- Delete account

# Tech Stack
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Testing: Postman
